**Lighthouse** 是 Google 开发的一个开源网站性能审计工具，内置在 Chrome 浏览器中。

### 主要功能

- **性能评分** (Performance) - 加载速度、交互响应
- **无障碍性** (Accessibility) - 对残障用户的友好程度
- **最佳实践** (Best Practices) - 安全性、代码质量
- **SEO** - 搜索引擎优化
- **PWA** - 渐进式 Web 应用检查

### 如何使用

1. 打开 Chrome 浏览器
2. 按 `F12` 打开开发者工具
3. 切换到 **Lighthouse** 标签页
4. 点击 **Analyze page load** 生成报告

### 或者在线使用

访问 [PageSpeed Insights](https://pagespeed.web.dev/) 输入网址即可测试

它会给出 0-100 的分数，并提供具体的优化建议（比如压缩图片、减少 JavaScript 等）。是检测网站性能最常用的工具之一！
